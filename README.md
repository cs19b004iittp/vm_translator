# VM Translator
![](https://github.com/josephan/vm_translator/workflows/Python%20package/badge.svg)

A Jack VM code to HACK assembly translator (from NAND2Tetris) written in python3.

## Usage

```bash
$ python3 VMTranslator.py BasicTest.vm
# => BasicTest.asm
```
